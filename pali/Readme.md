### Setting
  - Set url of USER_INVITE_URL_ALLOW_LIST and PASSWORD_RESET_URL_ALLOW_LIST in the config.json.

### Secrets
  Create file with no terminators `echo -n "mypassword" > _secrets/passwd` 

### Uploads
